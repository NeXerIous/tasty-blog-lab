# Tasty Blog 🍳

**Tasty Blog** — это современный кулинарный веб-портал, где пользователи могут находить рецепты, читать статьи о еде и вдохновляться на кулинарные шедевры. Проект разработан в рамках лабораторной работы по дисциплине «Интернет-технологии и веб-программирование».

## 🚀 Функционал

- **Адаптивная верстка:** Корректное отображение на различных устройствах.
- **Интерактивное меню категорий:** Фильтрация (визуальная) рецептов без перезагрузки страницы.
- **Анимации:** Плавное появление контента при скролле и зум-эффекты на карточках.
- **Семантический HTML5:** Использование тегов header, main, section, article для лучшей структуры.

## 🛠 Технологии

В проекте использован современный стек веб-разработки:

- **HTML5 & CSS3** (Flexbox, CSS Variables, CSS Transitions)
- **JavaScript (ES6+)** (DOM Manipulation, Event Listeners)
- **Node.js & npm** (Управление зависимостями)
- **Figma** (Проектирование интерфейса)
- **Git & GitHub** (Контроль версий)

## 📦 Установка и запуск

Чтобы запустить проект локально:

1. Клонируйте репозиторий:
   git clone https://github.com/NeXerIous/tasty-blog-lab.git

2. Перейдите в папку проекта:
   cd tasty-blog-lab

3. Установите зависимости:
   npm install

4. Откройте index.html через Live Server в VS Code.
